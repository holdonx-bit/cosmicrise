# 🪨 CosmicRise

Браузерна космічна гра виживання. Починаєш з пилинки — фінішуєш чорною дірою.

## Як грати

Поглинай об'єкти менші за себе, уникай більших. Чим більша маса — тим далі до наступної стадії.

**Управління:** WASD / стрілки / миша

## Стадії (14)

Космічний пил → Камінець → Камінь → Валун → Метеорит → Астероїд → Карликова планета → Планета → Газовий гігант → Коричневий карлик → Зірка → Гігантська зірка → Надгігант → **ЧОРНА ДІРА**

## Події

- ☄️ **Метеоритний дощ** — хвиля швидких метеоритів
- 💥 **Наднова** — ударна хвиля що пошкоджує все навколо
